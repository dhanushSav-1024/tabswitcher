# TabSwitcher

A Firefox extension for switching tabs fast using the keyboard.

## What it does

- Press **Alt+Space** to open the switcher
- Type to search tabs by title or URL
- **↑↓** to navigate, **Enter** to switch, **Esc** to close
- **Backspace** (empty search) to close a tab
- **Ctrl+Enter** to open your query as a new Google search tab
- Tabs sorted by most recently used
- Click the ⚙ icon to remap the open shortcut
